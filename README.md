# DevinOM NVim Configs

## Prerequisites
- Homebrew
- NeoVim

## First time setup
Clone Packer 
```bash
git clone --depth 1 https://github.com/wbthomason/packer.nvim\
 ~/.local/share/nvim/site/pack/packer/start/packer.nvim
```
Install ripgrep
```bash
brew install ripgrep
```
Activate nvim
```bash
nvim .
```
Run withing nvim
```bash
:PackerSync
```
